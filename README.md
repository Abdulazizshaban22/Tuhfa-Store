# Tuhfa Store

<div dir="rtl">

## متجر تحفة

`Tuhfa-Store` مستودع لمفهوم متجر إلكتروني باسم **تحفة**. عند التوثيق لم يظهر في الجذر ملف تشغيل مثل `package.json` أو `index.html`، لذلك هذا README يعرّف المشروع بشكل تأسيسي ومحافظ إلى أن تضاف البنية الفعلية أو تُنقل من مسار داخلي.

## الرؤية المقترحة

تحفة يمكن أن يكون متجرًا رقميًا للهدايا، المنتجات المختارة، التحف، أو التجارب التجارية ذات الطابع المحلي. التركيز المقترح:

- كتالوج منتجات واضح.
- صفحات تفاصيل منتج غنية بالصور والوصف.
- سلة وطلب شراء.
- تصنيفات ومجموعات موسمية.
- تجربة عربية RTL أولًا.
- تكامل لاحق مع دفع وشحن وإشعارات.

## نطاق وظيفي مقترح

| المجال | الوصف |
|---|---|
| Catalog | عرض المنتجات والتصنيفات. |
| Product Page | تفاصيل المنتج، الصور، السعر، والتوفر. |
| Cart | سلة شراء قابلة للتعديل. |
| Checkout | بيانات العميل، العنوان، الدفع، وتأكيد الطلب. |
| Orders | متابعة الطلبات وحالتها. |
| Admin | إدارة المنتجات والطلبات والمخزون. |
| Marketing | عروض، كوبونات، ومجموعات موسمية. |

## الحالة الحالية

- المستودع على فرع `main`.
- لا يوجد README سابق في الجذر.
- لم يظهر ملف `package.json` في الجذر عند التوثيق.
- لا توجد أوامر تشغيل مؤكدة حتى الآن.

## خطوات تالية مقترحة

1. تحديد نوع المنتجات والجمهور المستهدف.
2. اختيار البنية التقنية: Next.js، Vite/React، أو منصة تجارة جاهزة.
3. إضافة نموذج بيانات للمنتجات والطلبات والعملاء.
4. إضافة README تقني جديد بعد توفر الكود الفعلي.
5. إضافة سياسة خصوصية وشروط استخدام قبل أي نشر عام.
