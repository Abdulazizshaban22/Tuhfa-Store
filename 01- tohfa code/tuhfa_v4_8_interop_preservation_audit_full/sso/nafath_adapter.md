
# Nafath SSO Adapter — Conceptual Flow (Stub)

هذا المستند يوضّح آلية التكامل مع **نفاذ الموحّد (Nafath)** على مستوى التدفق:
1) يضغط المستخدم "الدخول عبر نفاذ".
2) منصتك ترسل طلب مصادقة إلى بوابة النفاذ وتعرض رمز المطابقة.
3) المستخدم يوافق داخل تطبيق نفاذ → يعود Callback بحالة النجاح/الفشل.
4) تُنشئ جلسة محلية (JWT/Session) وتربطها بالمستخدم.

> الرجوع إلى الأدلة الرسمية (my.gov.sa / SDAIA) لتسجيل التطبيقات وآليات الربط.
